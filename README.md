# Resume Builder

A personal resume PDF generator built with React, TypeScript, and Vite.

## Features

- Live preview as you type
- Creative, colorful template with sidebar layout
- Multiple roles per company (promotions)
- Tech stack tags per role
- Clickable contact icon links
- Export / import resume data as JSON
- Print to PDF via browser print dialog

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS (app shell)
- Custom CSS (resume template, print-safe)
- react-to-print

## Usage

```bash
npm install
npm run dev
```

Fill in your details in the form panel. Use **Save JSON** to export your data and **Load JSON** to restore it later. Click **Download PDF** to print.
